# Project

Static HTML site. No build tools, tests, or package manager.

## Files

- `index.html`, `members.html`, `kpmedia.html`, `portfolio.html`, `kpelectronics.html` — standalone pages
- `assets/` — third-party libs (Alpine.js, Tailwind CSS via CDN, Lucide icons, Fontshare)
- `style.css`, `variable.js` — shared assets

## Development

Open any `.html` file directly in a browser. No build step required.
